# Sudoku Prime — premium redesign, все уровни

## Что исправлено в этой версии

1. Убрана главная проблема со скрина: декоративные предметы больше не привязаны к `board-shell` и не могут залезать поверх поля судоку.
2. Поле стало чистой игровой зоной: декор находится в отдельном background/props layer и лежит ниже интерактивного UI по z-index.
3. PNG-ассеты подключены аккуратно:
   - `Утреннее кафе`: чашка кофе + круассан;
   - `Тихая библиотека`: старинные книги + антикварная ручка.
4. Для `Вечернего ливня` оставлен более дорогой минимализм: фон, дождь, стекло, свет, без крупных предметов около поля.
5. Для уровней 4–10 добавлены безопасные тематические CSS-декорации по краям экрана, чтобы все 10 уровней визуально отличались, но не мешали игре.
6. Добавлены адаптивные правила: на маленьких экранах лишний декор скрывается или уходит за край, чтобы не давить на поле.
7. Сборка и линт проверены:
   - `npm run lint` — без ошибок;
   - `npm run build` — успешно.

## Главное правило дизайна теперь

Поле судоку — зона без декора. Любые предметы, световые эффекты и декоративные элементы могут быть только:

- за полем;
- у краёв экрана;
- под интерактивным UI;
- скрыты/уменьшены на маленьких экранах.

## Что хорошо в приложении

- Хорошая функциональная база: генератор, заметки, подсказки, undo, таймер, пауза, сохранение прогресса.
- Есть PWA/offline-first, то есть приложение можно установить и играть без сети.
- Есть 10 миров, достижения, daily challenge, статистика и два языка.
- Идея уровней-миров сильная: она отличает приложение от обычных Sudoku-приложений.

## Что ещё можно улучшать дальше

- Доработать онлайн-режим до настоящего real-time multiplayer.
